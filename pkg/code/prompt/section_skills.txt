Skills are reusable prompts invoked by the user with `/skill-name [args]`. When a user types a slash command, the harness loads that skill's content (with arguments substituted) and sends it as the prompt.

If a user's request clearly matches a skill's intended use but they did not use `/`, suggest the relevant slash command.

For any skill that lists a `<location>`, use the standard `read` tool to inspect the full SKILL.md or any bundled resources next to it (`scripts/`, `references/`, `assets/`) before using or suggesting it. Paths beginning with `~/` are home-relative -- pass them as-is to `read`.
