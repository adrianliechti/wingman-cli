Use Language Server Protocol (LSP) to get code intelligence from the project's language server.

Supported operations:
- diagnostics: Get compiler errors, warnings, and hints for a single file (requires file)
- workspaceDiagnostics: Get all compiler errors and warnings across the entire project (no file needed)
- definition: Find where a symbol is defined (requires file, line, column)
- references: Find all references to a symbol (requires file, line, column)
- hover: Get type information and documentation for a symbol (requires file, line, column)

LSP servers are auto-detected based on project files (go.mod, package.json, Cargo.toml, etc.).
