# Wingman AI CLI

## Installation

#### MacOS / Linux

[Homebrew](https://brew.sh)

```
brew install adrianliechti/tap/wingman-cli
```

#### Windows

[Scoop](https://scoop.sh)

```shell
scoop bucket add adrianliechti https://github.com/adrianliechti/scoop-bucket
scoop install adrianliechti/wingman-cli
```
